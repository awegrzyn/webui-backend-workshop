# Linting

### Tasks

Lint and fix `index.js` file:
1. Run `npm install` to install `eslint`
2. Look at `.eslintrc.js` file that contains all the rules
3. Lint `index.js` file: `./node_modules/.bin/eslint index.js`
4. Fix all reported errors

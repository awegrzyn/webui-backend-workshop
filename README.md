# WebUI Backend Workshop

## Software requiremets
- NodeJS > `14`
- MariaDB database (optional)
- Browser with WebSockets debugging tool (eg. Firefox or Chrome)

## Skill requirements
- Usage of `node` and `npm`
- Basic knowleadge of MySQL
- Concepts: WebSockets, HTTP
